## Run the app locally

### With Docker
```
docker run -p 3000:3000 -p 5000:5000 -d sweesenkoh/failedquiz1_fiver_app:latest
```

or


### Manual setup

Install all dependencies
```
yarn setup
```
Start both frontend and backend
```
yarn start
```
